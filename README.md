# GenAI Information Extraction & Language Translation

## Description
This project demonstrates how to use Generative AI for information extraction and language translation. It showcases how structured data can be extracted from unstructured text and translated into different languages using prompt-based techniques.

## Features
- Extract structured information from raw text
- Perform language translation using AI
- Prompt-based data parsing
- Multi-language support
- Hands-on examples using AI models

## Tech Stack
- Python
- OpenAI API
- Jupyter Notebook

## Concepts Used
- **Information Extraction**: Converting unstructured text into structured data (e.g., name, date, entities).
- **Language Translation**: Translating text from one language to another using AI models.
- **Prompt Engineering**: Designing inputs to guide extraction and translation tasks.
- **Text Processing**: Handling and transforming raw text data.
- **LLM Capabilities**: Leveraging AI models for understanding and generating human language.

## Models Used
- **Text Models (GPT)**: Used for both extraction and translation tasks.
  - Examples: `gpt-4`, `gpt-4o`, `gpt-5`

## Installation Steps
1. Clone the repository:
   ```bash
   git clone <your-repo-link>
   ```
2. Navigate to the project folder:
   ```bash
   cd genai-information-extraction-translation
   ```
3. Install dependencies:
   ```bash
   pip install openai
   ```

## Usage
1. Add your OpenAI API key:
   ```
   /keys/.openai_api_key
   ```
2. Run Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
3. Open and execute:
   ```
   information_extraction_and_language_translation.ipynb
   ```
